Challenge Name: Obedient Cat

Description: This file has a flag in plain sight (aka "in-the-clear").
The link is provided to access the file.

Solution:
Method 1: Downloading the File
- By clicking the link [flag], the file is downloaded, now check the contents
  of the file by opening the file in Notepad or any text editor.
- The flag is litterally written in plain sight! 
- Copy the line and submit it as the solution.


Method 2: By wget and cat Commands
- Copy the link of the file and download the file by wget command: wget <file-path>
- You can paste the link you copied in the webshell by right click and pasting it.
- The file was downloaded named as flag.1
- We can see the content of the file by using cat command: cat flag.1
